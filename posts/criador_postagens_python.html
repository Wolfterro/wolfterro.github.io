Criador de Postagens em Python!Postado em: 26/03/2017

Um simples programa feito em um final de semana que me ajuda a postar mais neste site!


Como faz um certo tempo que não posto nada neste site, decidi criar um simples programa que me poderia me auxiliar neste processo.


Antes de utilizar este programa, eu era forçado a editar manualmente uma página de teste para posts e depois verificar manualmente se a postagem estava correta ou se eu esqueci de fechar alguma tag ou algo assim.


Já não mais preciso disto, agora posso utilizar o Criador de Postagens:



Tudo bem, o programa pode até não ser tão bonito assim ou complexo demais como um Dreamweaver, mas é exatamente do que eu preciso para criar um post rápido e simples para o meu site.


Programado na linguagem Python e utilizando o PyQt4, foi um bom projeto de fim de semana e um "comeback" na programação em Python, utilizando de forma mais eficiente a programação orientada a objetos que a linguagem tem a oferecer.


Atualmente passo mais tempo estudando para a minha faculdade em Redes de Computadores e um pouco de programação em C#, que diga-se de passagem, é a linguagem de programação mais bonita e relativamente simples que já vi, no que se refere a sintaxe e o uso da programação orientada a objetos.


Tenho alguns projetos em C# no meu GitHub e pretendo criar mais alguns no futuro, mas nunca conseguirei achar uma linguagem mais simples e fácil de programar do que Python, especialmente na criação de interfaces gráficas (GUI) utilizando o PyQt.


Devo adicionar mais algumas funcionalidades a este programa no futuro mas por hora é exatamente do que eu preciso para criar posts rápidos. Mas talvez um realçador de sintaxe não seja tão ruim assim...


Link para o repositório: https://github.com/Wolfterro/Criador-De-Postagens
