Mais Alguns Aplicativos para AndroidPostado em: 27/08/2017

Aplicativos simples e úteis que me auxiliam todos os dias!


Há pouco mais de dois meses que não posto neste site, mas neste meio-tempo estive estudando um pouco mais sobre desenvolvimento de aplicativos para a plataforma Android. E enquanto estive fazendo isso, decidi criar mais alguns aplicativos úteis para mim ou que sejam de algum interesse para outras pessoas.


Alguns destes aplicativos, por mais simples que sejam, me ajudaram a entender um pouco mais sobre a plataforma, especialmente na hora de desenvolver múltiplas interfaces (Activities, como são chamadas), permissões, conectividade, I/O e muito mais.


Eu normalmente deixo disponível o código-fonte destes aplicativos aqui mesmo no GitHub. Por este motivo, quando eu esqueço de alguma implementação que fiz em algum destes aplicativos, eu vou até o repositório deles e os uso como referência na hora de criar outros aplicativos, e isto me auxilia muito no dia a dia.


Aqui estão alguns dos novos aplicativos que desenvolvi:


BusDroid RJ




Em uma postagem anterior, eu disse com total certeza que iria portar o programa GPBus, escrito em Python e feito para ser usado em um ambiente de linha de comando (CLI), para uma das plataformas mobile disponíveis.


Pois bem, o programa agora está disponível para a plataforma Android, API 14 em diante (Android 4.0 em diante) em forma de um simples aplicativo que permite dizer a localização das linhas de ônibus disponíveis na cidade do Rio de Janeiro, utilizando a API pública do Data.Rio para obter as localizações.


É possível tambem, como mostra nas imagens, verificar o número de ordem do ônibus, o seu trajeto, a hora em que foi feita a última atualização do GPS do ônibus e sua velocidade atual. É possível também mostrar para o usuário a sua localização (caso haja permissão), permitindo que ele possa determinar a distância entre ele e os ônibus mais próximos.


Este é, provavelmente, o aplicativo mais elaborado criado por mim até então. Pretendo ainda adicionar mais algumas funções, como mostrar a rota da linha de ônibus inserida pelo usuário e pequenas alterações na interface do aplicativo.


Link para o repositório: https://github.com/Wolfterro/BusDroid-RJ


4DownDroid



Similar a outro programa que fiz, este aplicativo possui a função de baixar arquivos do 4chan (apenas).


Desde que desenvolvi o Boards Image Downloader, tive interesse em portá-lo, mesmo que de uma forma ainda mais simplificada, para uma das plataformas mobile que uso (Android e Windows Phone). Poder baixar arquivos de imageboard pelo celular me pareceu bem cômodo.


Pois bem, acabei criando este simples aplicativo, também voltado para o Android 4.0 em diante.


Para usá-lo, basta inserir a URL do tópico desejado e verificar o tópico. Assim que o tópico for verificado, o aplicativo irá mostrar o número de postagens, a quantidade de imagens e vídeos (e o total de arquivos) e se o tópico está arquivado ou não. Depois deste processo, basta apenas baixar os arquivos e estes estarão localizados dentro da pasta do aplicativo, indicada no campo de Pasta de destino.


Link para o repositório: https://github.com/Wolfterro/4DownDroid


TermalDroid



Este aplicativo, se comparado com os outros, é bem mais simples e bobinho, mas serve de exemplo para o desenvolvimento de aplicativos para a plataforma Android. É um exemplo simples e fácil para iniciantes, ideal para aqueles que estão começando agora a explorar a plataforma.


Este aplicativo tem como função fazer a conversão de diferentes escalas termométricas (Celsius, Fahrenheit e Kelvin) entre sí. Basta inserir o valor desejado, a escala que este valor representa (De) e a escala para qual você deseja converter (Para) e pronto, basta apertar o botão Calcular e a tela do aplicativo será atualizada para mostrar o valor.


Apesar de muito simples, estes tipos de aplicativos me ajudam muito no dia a dia. Como citei anteriormente, se eu me esqueço de como implementar algo em um novo aplicativo, eu sempre verifico o código-fonte de aplicativos mais simples como este e isto facilita muito o meu trabalho na hora do desenvolvimento.


Link para o repositório: https://github.com/Wolfterro/TermalDroid


E por hora é isso! Ainda há muita coisa que preciso aprender, especialmente o uso e a integração de diferentes bibliotecas e implementações em um aplicativo, o uso de diferentes API's e explorar totalmente as capacidades da biblioteca Android.


Bem, para quem começou a desenvolver aplicativos Android há apenas 5 meses, os aplicativos mostrados acima não são tão ruins, vocês não acham? :P
