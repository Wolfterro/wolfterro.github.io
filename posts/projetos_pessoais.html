Projetos pessoais: alguns exemplosPostado em: 17/02/2016

Alguns exemplos de meus projetos no Github


Já faz um ano e meio que comecei a desenvolver páginas da web e pequenas aplicações em linguagens de programação simples, como Python. Não posso dizer que sou profissional neste ramo, ainda tenho muito que aprender sobre programação e desta maneira crio programas simples e com diversas funcionalidades, seja uma aplicação para o ambiente de trabalho ou um script simples para facilitar a minha vida.


Abaixo, você verá alguns de meus projetos:


Cadastramento 2016


Tinha decidido criar algo útil para o dia-a-dia, como uma ferramenta de trabalho que é utilizada constantemente mas que poucas pessoas dão valor, algo como um software de cadastramento pessoal simples e que pode ser utilizada para serviços mais básicos em alguma loja ou até mesmo numa pequena clínica médica.


Decidi então criar o Cadastramento 2016:



O programa é bem simples, mas ainda não é exatamente ideal para uso profissional. É escrito em Python2 e pode ser executado em qualquer máquina Linux (ou unix em geral), dependendo apenas do Python2 e do editor de texto 'Nano'. Um usuário experiente (ou que tenha ao menos um conhecimento simples da linguagem Python) poderá editar o programa para usar o editor de texto 'Vi' ao invés do Nano.


O programa não está compilado e pode ser executado diretamente via terminal e editado diretamente com um simples editor de texto.


Link para o repositório: https://github.com/Wolfterro/Cadastramento-2016


Youtube-DL-Script


Um dos programas que mais me impressionou quando comecei a usar a plataforma Linux no dia-a-dia, tanto para estudo quanto para uso cotidiano, foi um programa escrito em Python, aparentemente simples e que realizava uma função muito útil, o youtube-dl.


O programa possui uma série de comandos para baixar vídeos e músicas de sites como o Youtube e diversos sites de streaming, alguns destes comandos são bem compridos e eu não queria ter que ficar digitando um comando grande toda vez que quisesse baixar uma música. Decidi então criar o Youtube-DL-Script:



O script é bem simples, também desenvolvido em Python2 mas possui versões para Python3, Ruby e Bash (Shell Script). Atualmente o script é voltado apenas para downloads de vídeos e músicas do Youtube, podendo talvez no futuro incluir outros sites de mídia compatíveis com o youtube-dl.


Outro requerimento é que um programa de conversão precisa estar instalado, seja ele o libav-tools ou o ffmpeg. O próprio youtube-dl irá se encarregar de usar um ou outro mas é necessário que esteja instalado caso queira converter um vídeo de um formato X para Y ou caso queira baixar uma música e queira também que o seu formato seja em MP3.


Como o script não é um programa standalone (isto é, não baixa os conteúdos por conta própria), ele necessita que o youtube-dl esteja instalado e seja reconhecido como um comando do sistema para que possa funcionar corretamente. Voltado também para a plataforma Linux (ou Unix em geral), sua versão em Python requer a versão correspondente instalada no sistema (Python2 ou Python3), assim também como a sua versão em Ruby.


Link para o repositório: https://github.com/Wolfterro/Youtube-DL-Script


Mega-Sena Simulator 2016


Comecei a programar em Java no início de 2015 por recomendação de um amigo e tenho achado a linguagem Java muito impressionante! Rica em diversas bibliotecas com diversas funções incluídas, a linguagem Java torna o desenvolvimento de certas aplicações um trabalho simples e totalmente descomplicado.


Mas como ainda sou iniciante na área de programação, decidi começar por algo mais simples, até mesmo algo bobo. Tinha criado então aplicações básicas descritas em tutoriais e vídeos na internet, como um conversor de temperatura e outros programas que não cheguei a publicar no Github.


Decidi criar então um joguinho simples, algo bobo para passar o tempo. Criei então o Mega-Sena Simulator 2016:



O jogo não é para ser algo muito sério, como apostas altas, micro-transações ou esquemas em geral, veja apenas como uma "paródia" e a falta de criatividade de um desenvolvedor iniciante.


As regras são bem simples, você precisa inserir uma combinação de seis números no campo de texto do programa (números abaixo de 01 e maiores que 60 não são válidos) e clicar no botão 'Jogar!'. O sorteio é feito automaticamente pelo programa e, se tiver sorte, você será milionário (de mentirinha, infelizmente) ou será requisitado a tentar novamente, se assim desejar.


Link para o repositório: https://github.com/Wolfterro/Mega-Sena-Simulator-2016


Em suma, sei que ainda tenho muito que aprender em relação a programação, não apenas outras linguagens mas como também suas convenções, suas funções e suas áreas de aplicação que mais se encaixam com suas características, como a linguagem C para aparelhos mais simples que necessitam do melhor aproveitamento de recursos possível e Java para ambientes mobile (como o Android) ou para aplicações que necessitem de portabilidade.


Espero poder criar outros programas e scripts no futuro. Programas mais úteis, mais eficientes e que seja necessário tanto para mim quanto para outras pessoas que estiverem interessadas em minhas criações.
